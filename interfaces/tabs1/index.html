HTML5

O HTML5 é a versão mais recente da linguagem HyperText Markup Language (HTML), ou Linguagem de Marcação de Hipertexto, e é fundamental para a internet atual.

CSS3

O CSS3 é a terceira versão do CSS, uma linguagem de estilo que permite criar a aparência de páginas da web, interfaces de usuário e sites.

JS

JavaScript é uma linguagem de programação que permite a criação de páginas web interativas, melhorando a experiência do usuário.

TypeScript

TypeScript é uma linguagem de programação de código aberto, desenvolvida pela Microsoft, que adiciona recursos à linguagem JavaScript.
